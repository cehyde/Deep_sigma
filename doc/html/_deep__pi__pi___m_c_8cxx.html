Detailed Description

lepto-production cross sections (electron or muon) based on `‘Angular distributions in hard exclusive production of pion pairs’', B. Lehmann-Dronke, A. Schaefer, M. V. Polyakov, and K. Goeke, PHYSICAL REVIEW D, (2001) 114001.

Input beam kinematics allow fixed target, head-on collisions, and crossing-angle collisions.
